# Ecommerce E2E Automation Framework

## Overview

This project is an End-to-End (E2E) test automation framework built using:

* Java
* Selenium WebDriver
* TestNG
* Maven
* Page Object Model (POM)
* Git & GitHub

This framework highlights an automated user experience on a E-Commerce website using Automation Exercise website. It follows the Page Object Model design to make the code easier to read, clearer and mainatinable. 

## Implemented Test Cases are as follows: 

* 1. Positive Login Test
* 2. Negative Login Test
* 3. Add Product to Cart
* 4. Remove Product from Cart
* 5. Add Multiple Items in Cart
* 6. Remove items from multiple cart
* 7. Checkout Cart Flow
